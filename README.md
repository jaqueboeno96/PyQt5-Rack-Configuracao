# PyQt5-Rack-Configuracao
Interface PyQt5 para configuração de racks com alocação de bobinas por cores e padrões específicos. Otimiza o processo de alocação de bobinas na produção, melhorando a eficiência do PCP (Planejamento e Controle da Produção). Facilita a organização visual e intuitiva das bobinas em racks.
